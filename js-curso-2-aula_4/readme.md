# 🎯 Jogo do Número Secreto

Jogo simples feito com HTML, CSS e JavaScript onde você precisa adivinhar um número aleatório.

## 🚀 Funcionalidades

* Número aleatório entre 1 e 100
* Dicas (maior ou menor)
* Contador de tentativas
* Reiniciar jogo

## ▶️ Como usar

1. Abra o arquivo `index.html`
2. Digite um número
3. Clique em **Chutar**

## 🛠️ Tecnologias

* HTML
* CSS
* JavaScript
